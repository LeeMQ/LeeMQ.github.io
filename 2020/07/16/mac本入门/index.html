macbook 入门

前面的话

　　第一次使用 Mac 之前，需要改变一些原有思维，不应该使用 Windows 的思维习惯去使用 Mac，Mac 会节省系统维护、清理杀毒、升级驱动等操作的时间，让我们可以专注做真正重要的事情，本文将详细介绍 macbook 入门的相关知识

按键
　　由于 macbook 简化了键盘，取消了标准键盘中的很多按键，下面是常用的补充按键
【delete】 fn + backspace
【home】 fn + 左箭头
【end】 fn + 右箭头
【pageup】 fn + 上箭头
【pagedown】 fn + 下箭头

电池使用
　　现有笔记本、手机，或是锂电池组成的电源系统均会带有电池保护板，过充保护是其中的基本功能之一，充满了就会断开充电回路；几乎上述电器的充电和自身用电都是双回路，也就是说充满后，电器是直接使用市电的，不会用电池；锂电池是没有记忆效应的，一次完整的充放电记为一次循环使用
　　macbookair用的是三元锂电池，在锂电池中寿命较短，约500次。所以，如果不一直插电源，那就是说反复使用电池，会发现一年后电池就大不如前了，而一直插着市电呢，至少不会过充
　　当然，锂电池在高温下满电是会影响寿命的，这里的高温指55摄氏度以上的温度，目前的数据显示持续一个月损失1-2%
　　为什么一直用市电，从不用电池，一年后容量仍然减少呢？一是随着时间推移电池自身的老化，第二是电池还有自放电，第三相信用macos的一般不会关机，而是合盖休眠，这也是耗电的，由此影响了电池寿命
　　苹果官方说明应当每个月做一次深放电，倒不是说要保养电池，锂电池没有记忆效应，是不需要定期深放的。但是电源系统对电池容量的校准，需要定期做循环才能校准
　　总结如下，平时插着电源使用，别忘了每月做一次深放电，用于校准电池容量

触摸板
　　macbook 的触摸板功能强大，使用好触摸板，可以大大提高工作效率，且可以不使用鼠标
　　点按： 左键点击效果
　　双指点按： 右键点击效果
　　双指轻点两下： 智能缩放
　　双指滚动： 滚轮效果
　　双指张开或捏合： 缩放效果
　　双指左扫或右扫： 前进或后退
　　双指从右侧边缘向左轻扫： 通知中心
　　三指拖移：（系统偏好设置 -> 辅助功能 -> 鼠标和触控版 -> 触控版选项 -> 启用拖移）
　　四指左扫或右扫： 在全屏幕的应用之间轻扫
　　五指张开或捏合： 显示桌面

调度中心
　　1、关闭仪表盘
　　系统偏好设置 -> 调度中心 -> 仪表盘
　　2、打开调度中心
　　四指上滑，或者使用快捷键command + 上箭头
　　3、显示桌面
　　快捷键 fn + f11
　　4、当页面被放大到全屏后，默认可以使用四指左滑或右滑来调出当前应用的页面
　　5、触发角
　　触发角位于调度中心中，可以设置触摸板的四个角为触发角，默认没有任何效果

快捷键
【截图快捷键】
　　shift + command + 3 将屏幕图片存储为文件
　　shift + command + 4 将所选区域的图片存储为文件
　　shift + command + 6 将触控栏的图片存储为文件
【窗口快捷键】
　　command + N 新建窗口
　　command + Q 关闭程序
　　command + H 隐藏最前面的应用的窗口
　　command + M 最小化
【浏览器快捷键】
　　command + T 新建标签页
　　command + W 关闭标签页
　　command + option + 左右箭头 前一个标签或后一个标签
　　command + option + i 打开或关闭控制台
　　command + R 刷新

软件设置
【软件安装】
　　1、下载软件到Mac之后双击运行，这时候就会看到桌面上生成了一个新的硬盘图标
　　2、同时还会弹出一个新的安装窗口，如果没有出现这个窗口，就去双击那个硬盘图标。要做的就是将软件的图标拖动到旁边的文件夹图标中
　　3、安装完成后最好还是打开Finder，在左侧找到那个新生成的硬盘，点击旁边的三角，这样就算是把硬盘弹出了
　　如果没有 applications 图标，则可以打开 “访达”，直接把 应用程序(applications)文件夹中

【软件卸载】
　　打开Finder，点击左侧的“应用程序”，然后把相应的软件直接拖到“废纸篓中”，这就算卸载了

常用功能
【分屏】
　　1、长按左上角的全屏按钮时，窗口会缩小，可以将其拖动到屏幕的一侧
　　2、松开该按钮，然后点按另一个窗口，以便开始并排使用这两个窗口
【utf8】
　　打开Txt文件时，提示未能打开文稿”***.txt”。文本编码”Unicode(UTF-8)”不适用
　　要解决上面的问题，需要进行以下几步：
　　1、打开mac自带的文本编辑器 textEdit.app， 搜索textEdit.app后打开即可
　　2、打开文本编辑程序的偏好设置，切换到“打开和存储”选项，在“纯文本文件编码”中设置“打开文件”的值为“中文(GB-18030)”
　　3、重新打开刚才的文本文件，就可以了
【活动监视器】
　　Windows中有任务管理器，可以用来查看本机运行的进程的状态和结束进程
　　Mac中的Activity monitor(活动监视器)有差不多的功能，在Spotlight中搜索 activity， 可以快速启动Activity monitor

声音bug
　　有时候 Mac 从睡眠状态恢复之后没有声音，这是 Mac OS X 系统的一个 Bug。这是因为 Mac OS X 的核心音频守护进程「coreaudiod」出了问题，虽然简单的重启电脑就能解决，但是如果此时开启了很多程序后者有其他情况不想重启电脑的话，有两种方法可解决此问题
　　1、在活动监视器窗口右上角的搜索框里输入audio，此时可以搜索到coreaudiod进程。选中coreaudiod进程，点击活动监视器窗口左上角的退出进程按钮，在弹出的对话框中点击退出。coreaudiod进程退出后会自动重启，这时声音就恢复了
　　2、在终端输入sudo killall coreaudiod命令并输入密码即可

外接显示器bug
　　当 mac 的外接显示器拔掉与 mac 的连接线，再次连接时，有时会出现连接不上的情况，可以采取以下措施解决此情况
　　系统偏好设置 -> 显示器 -> 排列 -> 勾选镜像显示器
　　当外接显示器显示正常后，再取消勾选镜像显示器即可

快速查看
　　只需要选中文件后按一下空格键，就能快速查看各类文件的内容，而不需要打开任何软件
